SATDBailiff Sample Validation
Collected with:
v0.7-beta
This is the dataset used to manually validate the performance of the tool that was originally defined in a study performed by Maldonado et al. [1]. This includes a random stratified sampling of 200 entries along with their classification of correctness. These instances are pulled from the "Maldonado Study Replication". Each correctness classification is accompanied by references to GitHub for each part of the instance output.
Maldonado Study Replication
Collected with:
v0.7-beta
The full dataset of the replicated Maldonado study [1]. This includes 5 projects from conception until their terminal commits noted here.
Large-Scale dataset
Collected with:
v1.0
A dataset collected for use in future works. Produced from a set of well-engineered projects as identified by Munaiah et al. [2]. A list of these repositories can be found here, where all removed or now-private repositories (as of 6/11/2020) have been removed.
[1]
E. D. S. Maldonado, R. Abdalkareem, E. Shihab and A. Serebrenik, "An Empirical Study on the Removal of Self-Admitted Technical Debt," 2017 IEEE International Conference on Software Maintenance and Evolution (ICSME), Shanghai, 2017, pp. 238-248, doi: 10.1109/ICSME.2017.8.
[2]
Munaiah, Nuthan & Kroh, Steven & Cabrey, Craig & Nagappan, Meiyappan. (2016). Curating GitHub for engineered software projects. 10.7287/PEERJ.PREPRINTS.2617.
